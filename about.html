About Us

This site is a collaboration between the Seamly Patterns and The Pattern Share project.

The Seamly Software(formerly Tau Meta Tau Physica) project was founded by Susan Spencer to meet a need for a digital pattern making software tool. The prototype was implemented as a batch-mode application written in Python and is currently being converted to an HTML5 web application.

The Pattern Share project is a masters thesis project by designer Justin Couch, who attends the Rhode Island School of Design.

Major code contributions to the the Seamly project and the collaboration with The Patttern Share were made by Steve Conklin, an open source software developer.

Additional web site development and styling was contributed by Molly Richardson, founder of MR 21 Solutions.
